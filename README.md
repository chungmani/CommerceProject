## 커머스 프로젝트 과제
카테고리별로 상품을 등록하고, 선택한 상품들을 보여주는 프로그램입니다.

# Product 클래스 역할
- 상품 정보 담당 (이름, 가격, 설명, 재고)
- 가격이나 재고는 0이 입력되면 예외발생

# CommerceSystem 클래스 역할
- 커머스 프로그램(시스템) 역할
- 카테고리 리스트를 생성하고, 카테고리를 추가함
- start() 함수로 프로그램 시작, 종료

# Category 클래스 역할
- 상품 리스트 관리
- 카테고리에 상품들을 추가함
- 등록된 상품들을 보여줌

# Customer 클래스 역할
- 고객 정보 담당

# 추가 예정
- 장바구니 및 주문하기 기능
- 관리자 모드
- Enum을 통한 고객등급별 할인 기능
- 람다&스트림을 활용한 상품 검색 및 관리 기능
